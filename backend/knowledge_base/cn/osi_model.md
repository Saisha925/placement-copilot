# Computer Networks: The OSI Model

The Open Systems Interconnection (OSI) model is a conceptual framework used to understand and standardize the functions of a telecommunication or computing system. It divides networking into seven layers:

1. **Physical Layer**: Transmits raw bit stream over the physical medium (cables, switches).
2. **Data Link Layer**: Defines the format of data on the network (MAC addresses, switches, framing).
3. **Network Layer**: Decides which physical path the data will take (IP addresses, routers, routing).
4. **Transport Layer**: Transmits data using transmission protocols including TCP and UDP (Port numbers, reliability).
5. **Session Layer**: Maintains connections and is responsible for controlling ports and sessions.
6. **Presentation Layer**: Ensures that data is in a usable format and is where data encryption occurs.
7. **Application Layer**: Human-computer interaction layer, where applications can access the network services (HTTP, FTP, SMTP).
